# WorldBorder

World border mod for [ElementZero](https://github.com/Element-0/ElementZero).<br><br>

Config
-------------
```
  WolrdBorder:
    enabled: true
    minX: -1000
    minZ: -1000
    maxX: 1000
    maxZ: 1000
    reachBorderMessage: You've reached world border!
    messageBorderApproach: You will reach border at %coord% pos in %number% blocks!
    informBefore: 10
```
